RSS Discovery

Read the News you Care About

What is RSS?

RSS stands for Really Simple Syndication and it is used as a way to convey information by converting the information to an XML file which can be easily read by a computer. The information contained within an RSS file is gathered by an RSS Reader (This site for example) and it converts the information into an easy to read format. RSS feeds allow you the ability to personalize your own news feeds to gather information from sources that you care about. This web app does not include any ads or popups so the only content you see is the content you choose.

Things to Note:

This application is programmed to parse the XML data from given feeds and display the data in a nice human friendly way. Different RSS feeds use different tags and different formats to organize their data. This program should be able to nicely parse data from a few different formats however some feeds that you add to your list may not display with pictures or have weird descriptions. This will be due to a different format that the program reads differently. This maybe fixed in the future depending on the popularity of the app. If you input a feed address into the app and it does not get added to your list, it will be because the address through an error which is preventing you from adding it.

How to Use This App

Step 1- Register for an Account

If you do not already have an account click the Register button to sign up for one. It is completely free.

Step 2- Login

Login to your account once you have registered!

Step 3- Find the web address of your favorite RSS feeds

Once you have logged you will be able to add some RSS feeds to you account. To do this find the web address of the feeds you want to add. (See how to find RSS feeds section to see how to do this). Once you have the address simply copy and paste it into the add feed field on the main screen. Assuming there are no mistakes your feeds should update with articles from the newly added feed.

Useful Feature:

By default this application will show articles from all of the feeds you have added. If you only want to read articles from specific sources then select the feed on the left sidebar to display only articles from that specific source. If you want to delete a feed from your news list click the 'X' symbol beside the feed you wish to delete. Note: This feature is not yet available for certain mobile devices.
